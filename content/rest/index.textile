The Ably REST client libraries provide a convenient and simple API to interact directly with the "Ably REST API":/rest-api/. All official client library APIs are consistent across every language, offer publishing and retrieving of historical messages, handle both basic and token authentication, work around unavailable data centres, include optional symmetrical encryption and provide access to channel, presence and metrics history.
